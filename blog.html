Question-1: What are the different ways to select an element in the DOM?

Some ways of select an elements are - getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll(). The usage of these depends on the requirement.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML will give you all the text with the tags. innerText will give you only text within the tag where you have used. And textContent is broader version of innerText, this will give you all the content including the hidden contents as well.

Question-3: What is event delegation in the DOM?

Event delegation is when you add events to the parent not the child. So that whenever you want to add any child after even after adding the event to the parent, this will work on them.

Question-4: What is event bubbling in the DOM?

Event bubbling is a scenario where the event bubbles all the way up to the dom tree. That means if you click a button, the parent div will also be clicked and the parents all the way to the body will be clicked. Its helpful for writing less code and work even after you add new child.

Question-5: How do you create, add, and remove elements using JavaScript?

You can create element by using createElement("div") or any element you want to created other than div. And then you have to add the elements into its parent element with appendChild. And if you want to remove elements you can use simply remove() after targeting the element by id or you can remove it by using removeChild.
